Introduction

Jvuetify is a html template to produce reproducible documents that are better presentable with additional features from Javascript fontend frameworks. This document is a sample template produced using Jvuetify which serves as it documentation.

Jvuetify is based on Vuetify.js. Vuetify is a Vue UI Library with beautifully handcrafted Material Components.

Features

  • HTML Tables with search and additional customizability
  • Figures with options to zoom in
    • Embed graphs from PlotlyJS.jl
    • Embed png’s from R’s ggplot2 using RCall.jl
  • Based on Pandoc to be able to generate table of contents ,stand alone documents ,katex equation rendering in HTML.
  • Tabs in the HTML document using the .tabset class
  • Highlight code using Prism.js
  • Latex equation rendering in HTML using katex
  • Document is broken down into sections using level 1 header

ggplot2 Figures

The following figures are created using ggplot2 package in R

R"library(ggplot2)"
jdata = rcopy(R"mpg")

rplot = """
        ggplot(df, aes(cty)) +
        geom_density(aes(fill=factor(cyl)), alpha=0.8) +
        labs(x="City Mileage", y="PDF",
              title="Density plot",
              subtitle="City Mileage Grouped by Number of cylinders",
              caption="Source: mpg",
              fill="# Cylinders") +
        scale_color_ipsum() +
        theme_ipsum_rc()
"""
Jvuetify.jggplot(rplot,jdata,"densplot.png")|>println

R"library(ggplot2)"
R"library(ggridges)"
R"library(viridis)"
jdata = rcopy(R"lincoln_weather")

rplot = """
        ggplot(df, aes(x = `Mean Temperature [F]`, y = `Month`, fill = ..x..)) +
        geom_density_ridges_gradient(scale = 3, rel_min_height = 0.01) +
        scale_fill_viridis(name = "Temp. [F]", option = "C") +
        labs(title = 'Temperatures in Lincoln NE in 2016') +
        theme_ipsum() +
        theme(
            legend.position="none",
            panel.spacing = unit(0.1, "lines"),
            strip.text.x = element_text(size = 8))
"""
Jvuetify.jggplot(rplot,jdata,"ridgeplot.png")|>println

PlotlyJS Figures

import PlotlyJS
function area1()
    trace1 = PlotlyJS.scatter(;x=1:4, y=[0, 2, 3, 5], fill="tozeroy")
    trace2 = PlotlyJS.scatter(;x=1:4, y=[3, 5, 1, 7], fill="tonexty")
    PlotlyJS.plot([trace1, trace2])
end
plotlygraph(area1(),"areagraph")
